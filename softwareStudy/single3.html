구조체

구조체(struct)

-C언어 자료형-

-기본 자료형(Primitive Data Type, Built-in Data Type)
C언어에서 제공하는 가장 단순한 자료형
Ex) int, double, char 등등

-파생 자료형
기본 자료형들을 이용해 만든 (또는 파생된) 자료형
같은 자료형의 변수를 모아 놓은 배열, 다른 변수를 가리키는 포인터 등등
Ex) 배열, 포인터 등

-사용자 정의 자료형 (User-defined Data Type)
프로그래머 스스로 정의한 자료형
서로 다른 자료형을 모아 놓은 구조체, 사용자가 목록을 정의해 사용하는 열거형 등등
Ex) 구조체, 열거형 등

-구조체 자료형 정의-

구조체를 정의하려면 우선 struct라는 키워드 다음 이름표(Name tag)가 붙음
-아래 예제에서 struct employee가 새로운 타입의 이름이 됨
-그 아래 여러 개의 변수를 묶어 employee라는 타입의 멤버 변수로써 사용됨
-구조체의 타입 정의하는 항상 세미콜론으로 끝남

구조체 멤버 변수

구조체를 이루는 요소 하나하나를 멤버(Member) 또는 멤버 변수(Member variable)라고 부름
-멤버가 모여 구조체를 이룸
-구조체 멤버 변수에 접근하려면 도트 연산자 (dot operator, member access operator) 사용

Ex)
struct employee buyer;
strcpy(buyer.name, "Jaehyung Eun");
buyer.age = 37;
buyer.dependents = 5;

구조체 멤버 변수도 메모리 상에서는 연속적으로 나열되어 있음
이를 이용해 마치 배열처럼 초기화 할 수도 있음

Ex)
struct employee buyer = {"Jaehyung Eun", 37, 5};

-구조체 포인터-

포인터가 가리키고 있는 공간에 구조체가 저장되어 있음

struct car {char name[8]; int year; int mileage;};
struct car mycar;
struct car *p = &mycar;
p = (struct car *)malloc(2 * sizeof(struct car));

구조체에 저장된 데이터가 클 경우, 이것을 직접 다루려면(복사나 함수 인자등) 그만큼 시간적, 공간적 부담이 생김
-포인터만 넘겨주면 훨씬 빠르고 효율적으로 데이터를 다룰 수 있음
